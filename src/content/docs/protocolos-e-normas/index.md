# Protocolos

Bem-vindo à seção de **Protocolos** da Base de Conhecimento da Secretaria de Saúde.

## O que você encontrará aqui

Esta seção contém protocolos clínicos, procedimentos operacionais padrão (POPs) e diretrizes técnicas que orientam as práticas dos profissionais de saúde.

## Categorias de Protocolos

- **Protocolos Clínicos**: Diretrizes para diagnóstico e tratamento
- **Procedimentos Operacionais**: POPs para procedimentos técnicos
- **Protocolos de Vigilância**: Orientações para vigilância epidemiológica
- **Protocolos de Emergência**: Procedimentos para situações de emergência

## Como usar

Navegue pelos documentos usando a barra lateral à esquerda. Todos os protocolos incluem informações de versão, status e data de última atualização.
